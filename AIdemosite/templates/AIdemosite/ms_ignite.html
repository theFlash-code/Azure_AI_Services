Project Bonsai (預覽)

Microsoft Bonsai Project is a low-code industrial AI development platform, where engineers will be able to create AI-powered automation without the need for an extensive background in data science. Companies are using Project Bonsai to help improve product quality and efficiency. Updates:

  • Support for action masking, which will help to guarantee valid behavior and speed up training by restricting the actions that AI can take.

    Action mask

    Action masks restrict the set of actions a learned concept can select, based on the current state.

  • Training from data with OfflineRL (Offline Reinforcement Lerning)

  • Model-based training acceleration reduce time and computational cost required to train a control policy on a key customer use case.

  • Assessment 2.0 offer rich analytical tools to help customers evaluate automation performance.

  • Bonsai Samples in Azure Marketplace

Project Bonsai (preview)

Microsoft Bonsai Project 是一個低代碼工業 AI 開發平台，工程師無需具備廣泛的資料科學背景即可創建AI驅動的自動化。公司正在使用 Project Bonsai 來幫助提高產品質量和效率。更新：

  • 支援動作遮罩，這將有助於通過限制AI可以採取的動作來保證有效行為並加快訓練速度。

    動作遮罩

    動作遮罩根據當前狀態限制學習概念可以選擇的動作集。

  • 使用 OfflineRL從資料中訓練 (離線強化學習)

  • 基於模型的訓練加速 減少在關鍵客戶訓練模型的時間和計算成本，。

  • 評估 2.0 提供豐富的分析工具，幫助客戶評估自動化性能。

通過使用 Power Platform 自動化重複流程

低代碼使人們和組織能夠在 IT 資源上以更低的成本做更多的事情，而且在革新日常流程方面，沒有什麼能比得上 Power Platform。Power Platform 使任何人都可以輕鬆構建應用程序和網頁、自動化工作流、創建虛擬代理和分析數據。在 Microsoft Ignite，我們很高興能分享新的創新，這些創新使 AI 和低代碼對所有技能水平的用戶都更加直觀。 在 GitHub Copilot 的成功基礎上，我們將自然語言功能引入 Power Automate，並提供對雲流的描述。現在，創客只需要用自然語言描述他們想要構建的流程，並生成相關流程的建議——然後創客添加連接以完成他們的流程。就這麼容易。在 AI Builder 中，反饋循環使您能夠標記和重新訓練文檔以提高準確性，現在還額外支持非結構化數據和文本識別以識別 164 種語言。 Microsoft Power Platform Blog

使用 Azure 應用 AI 和認知服務快速輕鬆地將 AI 嵌入到業務應用程序中

Azure 應用 AI 服務和 Azure 認知服務為開發人員提供了預構建和可自定義的模型，以快速部署智能應用程序。儘管在大流行期間人工智能的採用總體上有所增加，但它特別加速了諸如H&R Block的文檔處理、Beiersdorf的知識挖掘以及Progressive Insurance的銷售和服務場景中的自然語言理解等用例。這些公司正在使用 Azure AI 服務來消除關鍵業務流程中的摩擦，以幫助最終用戶和員工將時間和精力集中在有價值的工作上。 現在，客戶可以在Azure Form Recognizer Studio中使用新的預構建合同模型來提取實體，例如各方和付款條款以進行下游處理。Azure OpenAI 服務（預覽版）現在可以通過邀請訪問 DALL·E 2。Azure 語言認知服務現在提供擴展的摘要、聯絡中心功能、語言支持和 Language Studio 增強功能。 Microsoft AI Blog
